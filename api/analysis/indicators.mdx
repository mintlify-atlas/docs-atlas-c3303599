## Overview

The technical indicators endpoint calculates a comprehensive set of technical analysis indicators including moving averages, momentum oscillators, trend indicators, and volatility measures. These indicators form the foundation of the trading signal generation system.

## Query Parameters

<ParamField query="symbol" type="string" required>
  The trading pair symbol to analyze (e.g., "BTC/USDT").
  
  Example: `BTC/USDT`
</ParamField>

<ParamField query="exchange" type="string" required>
  The exchange name where the symbol is traded (e.g., "binance").
  
  Example: `binance`
</ParamField>

<ParamField query="timeframe" type="string" default="1h">
  The candlestick timeframe for analysis. Common values include:
  - `1m` - 1 minute
  - `5m` - 5 minutes
  - `15m` - 15 minutes
  - `1h` - 1 hour (default)
  - `4h` - 4 hours
  - `1d` - 1 day
</ParamField>

          - Price > SMA: Bullish trend
          - Price < SMA: Bearish trend
        </ResponseField>
        
        <ResponseField name="sma_50" type="number">
          Simple Moving Average with 50-period window. Longer-term trend indicator.
          
          **Golden Cross:** SMA20 crosses above SMA50 (bullish)
          **Death Cross:** SMA20 crosses below SMA50 (bearish)
        </ResponseField>
        
        <ResponseField name="ema_12" type="number">
          Exponential Moving Average with 12-period window. More responsive to recent price changes than SMA.
        </ResponseField>
        
        <ResponseField name="ema_26" type="number">
          Exponential Moving Average with 26-period window. Used in MACD calculation.
        </ResponseField>
        
        <ResponseField name="rsi_14" type="number">
          Relative Strength Index with 14-period window. Momentum oscillator measuring speed and magnitude of price changes.
          
          **Range:** 0 to 100
          
          **Interpretation:**
          - RSI > 70: Overbought (potential reversal down)
          - RSI < 30: Oversold (potential reversal up)
          - RSI 40-60: Neutral zone
        </ResponseField>
        
        <ResponseField name="macd" type="object">
          Moving Average Convergence Divergence indicator showing relationship between two EMAs.
          
          <Expandable title="MACD Components">
            <ResponseField name="macd" type="number">

            </ResponseField>
            
            <ResponseField name="signal" type="number">
              Signal line (9-period EMA of MACD). Used for crossover signals.
            </ResponseField>
            
            <ResponseField name="histogram" type="number">
              MACD histogram (MACD - Signal). Visual representation of momentum divergence.
              
              **Interpretation:**
              - Histogram > 0 and increasing: Strengthening bullish momentum
              - Histogram < 0 and decreasing: Strengthening bearish momentum
            </ResponseField>
          </Expandable>
        </ResponseField>
        
        <ResponseField name="bollinger_bands" type="object">
          Volatility indicator consisting of a moving average and two standard deviation bands.
          
          <Expandable title="Bollinger Bands Components">
            <ResponseField name="upper" type="number">
              Upper band (SMA + 2 standard deviations). Potential resistance level.
            </ResponseField>
            
            <ResponseField name="middle" type="number">
              Middle band (20-period SMA). Average price level.
            </ResponseField>
            
            <ResponseField name="lower" type="number">
              Lower band (SMA - 2 standard deviations). Potential support level.

          </Expandable>

## Indicator Calculations

<Info>
  **Data Requirements**
  
  Technical indicators require sufficient historical data:
  - Minimum 20 data points for basic indicators (SMA, Bollinger Bands)
  - Minimum 50 data points recommended for reliable calculations
  - More data points = more accurate long-term indicators (SMA50, etc.)
  
  The system retrieves the last 50 candlesticks by default from the database.
</Info>

## Supported Indicators

NeuraTrade calculates the following technical indicators:

### Moving Averages
- **SMA (Simple Moving Average):** 20, 50 periods
- **EMA (Exponential Moving Average):** 12, 26 periods

### Momentum Oscillators
- **RSI (Relative Strength Index):** 14 periods
- **Stochastic Oscillator:** K=14, D=3 periods

### Trend Indicators
- **MACD:** Fast=12, Slow=26, Signal=9
- **Support/Resistance Levels:** Derived from recent highs/lows

### Volatility Indicators
- **Bollinger Bands:** Period=20, StdDev=2.0
- **ATR (Average True Range):** 14 periods

### Volume Indicators
- **OBV (On-Balance Volume):** Cumulative volume flow

<Note>
  All indicator calculations use the **goflux** library (go-talib wrapper) for accurate, industry-standard technical analysis.
</Note>

## Understanding the Indicators

<Accordion title="Simple vs Exponential Moving Averages">
  **SMA (Simple Moving Average):**
  - Calculates arithmetic mean of prices over N periods
  - All prices weighted equally
  - Smoother but slower to react to price changes
  - Better for identifying long-term trends
  
  **EMA (Exponential Moving Average):**
  - Gives more weight to recent prices
  - More responsive to new information
  - Better for short-term trend identification
  - Used in MACD calculation
</Accordion>

<Accordion title="RSI Divergence">
  RSI divergence can signal potential reversals:
  
  **Bullish Divergence:**
  - Price makes lower lows
  - RSI makes higher lows
  - Suggests weakening bearish momentum
  
  **Bearish Divergence:**
  - Price makes higher highs
  - RSI makes lower highs
  - Suggests weakening bullish momentum
</Accordion>

<Accordion title="Bollinger Band Squeeze">
  When Bollinger Bands narrow (low volatility), it often precedes a significant price move:
  
  - **Squeeze:** Bands narrow significantly
  - **Breakout:** Price breaks above/below bands with expanding width
  - **Direction:** Use other indicators (RSI, MACD) to predict breakout direction
</Accordion>

## Best Practices

<Tip>
  **Combine Multiple Indicators**
  
  No single indicator is perfect. Best practice is to:
  - Use 2-3 indicators from different categories
  - Confirm signals across multiple timeframes
  - Consider volume indicators for confirmation
  - Monitor indicator divergences
</Tip>

<Warning>
  **Lagging vs Leading Indicators**
  
  Most technical indicators are **lagging** - they confirm trends after they've started:
  - **Lagging:** SMA, EMA, MACD (good for confirmation)
  - **Leading:** RSI, Stochastic (good for early signals, more false signals)
  
  Use a mix of both for balanced analysis.
</Warning>
